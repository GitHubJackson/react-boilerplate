# React-spa-template

基于最新版 `create-react-app` 搭建的单页面项目模板，集成了以下依赖

- Typescript
- webpack 5
- react 17
- React Router v6
- CSS Modules
- Less
- axios
- ahooks

## 特性

- 开箱即用的项目模板
- 代码风格统一（eslint/stylelint/prettier/commitlint/husky/EditorConfig）
- 环境配置
- 其他
  - 路径 alias
